Platform Configuration

Platform Name

Allow Custom Database

Maintenance Mode

Platform Database

The default database used by all user spaces

How Platform Database Works

The platform database credentials are embedded in database.js. All users' spaces are stored in this project unless they opt in to their own DB. To change the platform database, update the credentials in the source code and redeploy.

Active Configuration

Recommended Firestore Security Rules

Apply these rules in your Firebase Console → Firestore → Rules to allow the admin portal to write platform configuration:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // User spaces — read/write open (Phase 1)
    match /spaces/{spaceId} {
      allow read, write: if true;
    }
    // Platform config — readable by all, writable by all (Phase 1)
    match /config/{document} {
      allow read, write: if true;
    }
    // Subcollections (loves/hates, memories, events)
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
